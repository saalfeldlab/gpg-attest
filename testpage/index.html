gpg-attest — Manual Test Page

Use this page to test the browser context menu signing flow locally.

Serve over HTTP — not file://
The extension's service worker uses fetch(), which is blocked on file:// origins. Run a local HTTP server from the testpage/ directory: 
cd /workspace/testpage
python3 -m http.server 8080
# then open http://localhost:8080

Instructions

  1. Load the extension in Chrome (chrome://extensions, Developer mode, Load unpacked → extension/) or Firefox (about:debugging).
  2. Make sure the native host is installed: make install from /workspace.
  3. Open DevTools (F12) and switch to the Console tab.
  4. Right-click any image below to open the browser context menu.
  5. Click Select signing key…, select a key, then click Save.
  6. Right-click the image again and click Sign ✓ — the Console should print the sha256 hash and PGP signature.
